# Auth Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire up better-auth with email/password sign-up, sign-in, and sign-out, protected by a proxy-based session check, with a styled auth UI matching the design system.

**Architecture:** better-auth runs as a catch-all API route handler at `/api/auth/[...all]`. Session reads in server components and the proxy use `auth.api.getSession({ headers })`. Client components call `authClient.signIn.email()` / `signUp.email()` / `signOut()` from a shared browser client. The `(auth)` route group gives sign-in and sign-up a shared dark-fantasy shell layout without duplicating markup.

**Tech Stack:** better-auth 1.5.6, Drizzle ORM (drizzle adapter), Next.js 16 App Router, `proxy.ts` for route protection, Tailwind CSS v4, Spectral (Google Fonts via `next/font`)

> **Note:** The design spec lists `src/app/proxy.ts` but Next.js 16 requires the proxy file at `src/proxy.ts` (same level as `src/app/`). This plan uses the correct location.

- [ ] **Step 3: Create `src/proxy.ts`**

  ```ts
  import { NextResponse } from "next/server";
  import type { NextRequest } from "next/server";
  import { auth } from "@/lib/auth/server";

  export async function proxy(request: NextRequest) {
    const { pathname } = request.nextUrl;
    const session = await auth.api.getSession({ headers: request.headers });

    const isProtected = pathname.startsWith("/dashboard");
    const isAuthPage =
      pathname.startsWith("/sign-in") || pathname.startsWith("/sign-up");

    if (isProtected && !session) {
      return NextResponse.redirect(new URL("/sign-in", request.url));
    }

    if (isAuthPage && session) {
      return NextResponse.redirect(new URL("/dashboard", request.url));
    }

    return NextResponse.next();
  }

  export const config = {
    matcher: ["/dashboard/:path*", "/sign-in", "/sign-up"],
  };
  ```
